LanOnasis Memory Suite provides a unified memory layer for applications and agents. It is accessible via REST, SDKs, CLI, and MCP tools.

## Capabilities

- Store and retrieve structured memory items
- Semantic search and filtering
- Namespaces and access controls per project/user
- Observability and auditing

## Access paths

- REST: `https://api.lanonasis.com/api/v1/memory/...`
- MCP tools: via `https://mcp.lanonasis.com`
- SDK: `@lanonasis/memory-client`
- CLI: `@lanonasis/cli`
